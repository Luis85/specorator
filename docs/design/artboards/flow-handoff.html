the small stuff

Micro-interaction notes

timing
Total target: ~1 second

200ms confirm reveal · 100ms snapshot · 100ms file mutation · 250ms cross-fade · 350ms settle. Anything faster feels like a glitch; anything slower feels like a wait.

keyboard path
No mouse needed

⌘⏎ from anywhere in the cockpit triggers hand-off. ⏎ confirms. esc cancels. ⌘Z within 24h undoes — exactly like text editing.

sound
Silent.

No success chime. No haptic. The visible cross-fade is enough. Sound would feel like cheerleading — see principle 5.

repeat hand-off
Power-user mode

Once a user has handed off N times (TBD ~10), the inline confirm becomes opt-in. ⌘⏎ goes straight to mutation. Undo banner still shows. They've earned the speed.

Open question OQ-3 · gates auto-pass or explicit sign-off?

If auto-pass, this flow is what we get — gate ticks turn the button green and hand-off proceeds. If explicit sign-off, we need an extra "I sign off on this gate" frame between 01 and 02. Tradeoff: ceremony vs friction. Leaning auto-pass for solo users, explicit when team-mode is enabled. To resolve in DR-008.
