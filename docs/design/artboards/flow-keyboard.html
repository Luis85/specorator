catalogue · keyboard & command palette

Keyboard vocabulary — every command, shortcut, and context

Specorator's full command surface. Four groups: navigation, feature actions, gate & lifecycle, and utility. All commands are available via Obsidian's command palette (⌘P). Keyboard shortcuts are opt-in defaults — users can rebind any of them in Obsidian's hotkeys settings.

design constraint

Specorator registers no global keyboard shortcuts by default — Obsidian's keyspace is shared with dozens of plugins and the user's own bindings. The only default shortcut is ⌘ ⏎ for "confirm / advance" within Specorator-owned modal surfaces (hand-off confirm, gate cross, close feature). All other shortcuts are suggested defaults the user must opt into via Hotkeys settings.

choice
No default shortcuts except ⌘ ⏎

Obsidian's keyspace is contested. Registering shortcuts without asking causes conflicts with other plugins. The only default is ⌘ ⏎ on Specorator-owned confirm surfaces — a context where no other plugin will interfere. Everything else is opt-in.

choice
All commands in the palette

Every Specorator action is reachable via ⌘P. This is the contract. No action should require knowing a keyboard shortcut or finding a hidden button. The palette is the universal escape hatch — always there, always complete.
